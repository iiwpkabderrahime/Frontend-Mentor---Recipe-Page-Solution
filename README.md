# Recipe Page

A clean and responsive **Recipe Page**, built as a Frontend Mentor challenge using **HTML5 and CSS3**.

## 🚀 Overview

## 🔗 [Project Live Link](https://frontend-mentor-recipe-page-solutio-eight.vercel.app/)

This project recreates a beautiful recipe layout focused on **clean typography, spacing, and readable content structure**.

It includes:

* A recipe hero image
* Preparation time section
* Ingredients list
* Step-by-step instructions
* Nutrition table
* Fully responsive layout

---

## 🛠️ Built With

* **HTML5** semantic elements
* **CSS3** media queries
* **Flexbox**
* **Google Fonts**
* **Responsive design principles**

---

## 📱 Responsive Design

Optimized for:

* **Mobile** → full-width card layout
* **Tablet** → better spacing and typography scaling
* **Desktop** → centered recipe card with rounded corners

Breakpoints used:

* `max-width: 375px`
* `max-width: 1024px`
* `min-width: 1024px`
